# 🚀 Codeforces Analyzer

A Streamlit web application that analyzes any Codeforces profile using the official Codeforces API.

## 🌐 Live Demo

https://codeforces-analyzer-8ryyo4bxbyce65jv93qdza.streamlit.app/

## Features

- 👤 Profile Information
- 📜 Submission History
- 📈 Rating Progress Graph
- 🏆 Contest History
- 🏷️ Tag Analysis
- ✅ Correct Problems Solved
- 📊 Rating-wise Question Distribution

## Tech Stack

- Python
- Streamlit
- Pandas
- Requests
- Plotly

## Installation

```bash
git clone https://github.com/SinhaPushkar06/CodeForces_Analyzer.git
cd CodeForces_Analyzer
pip install -r requirements.txt
streamlit run app.py
```
